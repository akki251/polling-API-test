## Run Locally

Clone the project

```bash
  git clone https://github.com/akki251/polling-API-test.git
```

Go to the project directory

```bash
  cd polling-API-test
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Environment Variables
