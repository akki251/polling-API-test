## Run Locally

Clone the project

```bash
  git clone https://github.com/akki251/Placement-Cell---Skill-Test.git
```

Go to the project directory

```bash
  cd Placement-Cell---Skill-Test
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Environment Variables
